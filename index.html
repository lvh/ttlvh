Usage

ttlvh requires pandoc and lessc to build. From the root of your git repository, run:

typewriter -t git://github.com/lvh/ttlvh.git

If you have hub installed, you can use the following shorthand:

typewriter -t lvh/ttlvh

Consult the typewriter documentation for more information on how to use or build a theme. This theme should be a good starting point.
